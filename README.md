<h1 align = 'center'> Docker with mongoDB </h1>

1. [Install docker](https://www.docker.com/get-started)
2. git clone
3. cd note-docker-node-mongo
4. cd backend
5. run docker-compose up
6. open the app on [localhost](http://localhost/)
